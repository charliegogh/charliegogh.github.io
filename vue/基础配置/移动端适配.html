    # 适配插件

    amfe-flexible:动态修改font-size

    postcss-px2rem-exclude:解决引入第三方ui库出现样式变小的问题

    在vue.config.js中添加

    css: {
    loaderOptions: {
      postcss: {
        plugins: [
          require('postcss-px2rem-exclude')({
            remUnit: 75,
            exclude: /node_modules|folder_name/i
          })
        ]
      }
    }
  }
