  • 前端页面性能优化

  1. 引入jsdk(key,UI库)
<script type="text/javascript" src="http://webapi.amap.com/maps?v=1.3&key=您申请的key值"></script> 
<script src="https://webapi.amap.com/ui/1.0/main.js?v=1.0.11"></script>
  1. 增加webpack配置 直接引入cdn
 configureWebpack: {
    externals: {
      'AMap': 'AMap' // 高德地图配置
       'AMapUI': 'AMapUI'// ui组件库
    }
  }
  1. 组件中引入
import AMap from 'AMap' // 引入高德地图
