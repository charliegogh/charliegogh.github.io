    # 登陆页面底部文字被弹起解决方案

    1. 获取默认屏幕高度和实时屏幕高度
    • 默认屏幕高度：docmHeight:document.documentElement.clientHeight
    • 实时屏幕高度：showHeight:document.documentElement.clientHeight
    1. 实时设置屏幕高度
    window.onresize = () => {
      return (() => {
        this.showHeight = document.body.clientHeight;
      })()
    }
    1. 监听屏幕高度变化，对底部文字实时动态显示和隐藏
    watch: {
    showHeight() {
      if (this.docmHeight > this.showHeight) {
        this.hidshow = false
      } else {
        this.hidshow = true
      }
    }
  }
