# 依赖第三方库

  • iconfont 字体图标库
  • animate.css css3动画类库
  • flexible.js 手机端适配解决方案lib-flexible
  • Jquery dom操作库，采用 1.9.1版本是为了兼容IE9+
  • Swiper 幻灯片库，采用3.4.2版本也是为了兼容IE9+，swiper4只兼容到IE10+
